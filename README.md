### JustLemmeDebug

[![Greasyfork](https://cdn.jsdelivr.net/gh/JMcrafter26/badges@main/src/assets/available/greasy-fork/cozy.svg)](https://greasyfork.org/en/scripts/)


## Installation Guide

### PC/Mac

1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Greasemonkey](https://www.greasespot.net/) in your web browser.
2. Click on the installation badge or open the userscript URL in your browser.
3. Follow the prompts to install the userscript.
4. The userscript should now be active on the specified websites.

### iOS

1. Install the [Userscripts](https://apps.apple.com/app/userscripts/id1463298887) app from the App Store.
2. In the extension menu, select "Userscripts" to install the script.
3. In the popup, click on the yellow "Install" banner at the top.
4. Confirm the installation by clicking "Install" again.
5. The userscript should now be active on the specified websites.

### Android

Firefox on Android has built-in support for extensions like Tampermonkey. You can follow the same steps as for PC/Mac to install userscripts.

## Usage

Once installed, the userscript will automatically run on the Makerworld website. You can access its features through the provided interface elements.
