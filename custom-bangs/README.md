# Custom DuckDuckGo !Bangs

This userscript allows you to add your own custom !bangs to DuckDuckGo without modifying the built-in ones. You can define a search URL and an example search term for each bang or import a list of bangs from a remote JSON file. The script also provides a user interface for managing your custom bangs and testing them before adding them to DuckDuckGo.

## Features

- Add custom !bangs with a search URL and example search term.
- Test your custom !bangs before adding them.
- Remote !bangs lists with automatic updates and priority handling.
- Import and export your custom !bangs as JSON.
- Manage your custom !bangs through a user-friendly interface.
- DuckDuckGo !bangs collision detection
