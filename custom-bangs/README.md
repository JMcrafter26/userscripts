# Custom DuckDuckGo !Bangs

This userscript allows you to add your own custom !bangs to DuckDuckGo without modifying the built-in ones. You can define a search URL and an example search term for each bang.

## Features

- Add custom !bangs with a search URL and example search term.
- Test your custom !bangs before adding them.
- Remote !bangs
- Import and export your custom !bangs as JSON.
- Manage your custom !bangs through a user-friendly interface.
