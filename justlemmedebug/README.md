# JustLemmeDebug

**Unblock developer tools and debugging capabilities!** This userscript blocks anti-debugging scripts on websites, allowing developers to inspect and debug web pages without interference. Based on the concept from [LemmeDebug](https://github.com/deeeeone/userscripts).
